# emucap 실행 검증 어댑터

빌드한 패치가 실제 게임 경로에서 소비되는지 확인할 때 [emucap](https://github.com/mcpads/emucap)을 선택할 수 있다. 설치법과 에뮬레이터별 절차는 공개 배포 문서를 따르며 이 저장소에 복제하지 않는다.

검증할 산출물의 강한 해시와 `emucap`이 보고한 ROM 식별 정보가 일치하는지 먼저 확인한다. 자연 플레이가 아닌 상태 불러오기, 메모리 변경, 입력 주입이나 루틴 호출을 사용했다면 개입 내용과 우회한 경로를 기록한다.

실행 결과에는 다음을 남긴다.

- 정확한 산출물 식별 정보
- 시작 조건과 관측한 시나리오
- 기대 결과와 실제 결과
- 스크린샷·추적·메모리 등 근거의 식별 정보
- 상태 개입과 그 결과가 증명하지 못한 범위

해시 일치는 증거가 어느 빌드에서 나왔는지만 확인한다. 시나리오의 대표성, 화면 품질과 배포 가능 여부는 대상 위험과 HITL 판단으로 판정한다.

실행 자료는 제품 빌드 입력이 아니다. 저작권이나 로컬 환경 정보가 포함된 자료는 `evidence/private/`와 `.emucap/`에 두고, 공개 저장소에는 배포 가능한 요약과 식별자만 남긴다.
